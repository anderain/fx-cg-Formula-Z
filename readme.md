# FormulaZ

## (1)
样例 'a / b + c - 1'  
通过token自顶向下检查是否有syntax error 
## (2)
把token转换为expr_node，生成表达式树  
```
     [ / ]
    /     \
   /       \
[ a ]     [ + ]
         /     \
        /       \
     [ b ]     [ - ]
               /   \
              /     \
           [ c ]    [ 1 ]
```
## (3)
对表达式树进行排序
```
              [ - ]
             /     \
            /       \
         [ + ]    [ 1 ]
        /     \
       /       \
    [ / ]     [ c ]
   /     \
  /       \
[ a ]    [ b ]
```
## (4)
把表达式树转为渲染树  
```
       [___root___]
       /  /  |  \  \
      /  /   |   \  \
   [/] [+]  [c] [-] [1]
   / \
  /   \
[a]   [b]
```
